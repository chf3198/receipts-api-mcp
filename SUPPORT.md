# Support

## Getting Help

| Channel | Use for |
|---|---|
| [GitHub Issues](https://github.com/chf3198/receipts-api-mcp/issues) | Bug reports and feature requests (use the provided templates) |
| [GitHub Discussions](https://github.com/chf3198/receipts-api-mcp/discussions) | Questions, ideas, and general conversation |

## Before Opening an Issue

1. Check the [README](README.md) for local setup and usage instructions.
2. Search [existing issues](https://github.com/chf3198/receipts-api-mcp/issues) to see if the problem is already reported.
3. Verify the API is running locally with `uv run uvicorn receipts_api.main:app --reload` before reporting dashboard issues — the dashboard requires a live API.

## Response Time

This is a portfolio / interview project maintained by one person. Issues and discussions are monitored but response times may vary.
